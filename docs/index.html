How It Works

Up and running in three steps

Cloviz runs entirely on your machine. No data leaves your computer.

Install and start

Clone the repository and run the dev server. Cloviz starts watching your ~/.claude directory immediately.

Automatic indexing

Session files are parsed and indexed into a local SQLite database. Git commits are linked, costs are calculated, and patterns are detected.

Explore your data

Open the dashboard to browse projects, replay sessions, search conversations, and analyze your Claude usage patterns.

Quick Start

# Clone the repository
git clone https://github.com/nicobailon/cloviz.git
cd cloviz

# Install dependencies
bun install

# Start the dashboard
bun run dev
